# House Price Prediction using Linear Regression

## 📌 Project Overview
This project predicts house prices using machine learning.

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📊 Features Used
- GrLivArea (Square Footage)
- Bedrooms
- Bathrooms
- Overall Quality
- Garage Capacity
- Year Built

## 📈 Model
- Linear Regression

## 📉 Performance
- R² Score: ~0.8 (approx)
- RMSE: Moderate

## 📷 Output
Graph showing Actual vs Predicted prices.

## 🚀 How to Run
1. Install libraries: pip install pandas matplotlib scikit-learn
2. Run: python house_price.py

## 📁 Files
- house_price.py → Main code
- submission.csv → Predictions
